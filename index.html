FD

Floppy Disk Tool for N8VEM Project

Download this project as a .zip file Download this project as a tar.gz file

Summary

Application to test the hardware functionality of the Floppy Disk Controller (FDC) on the ECB DISK I/O, DISK I/O V3, ZETA SBC, Dual IDE w/ Floppy, or N8 board.

Binary Distribution

Pre-built binaries are available at https://github.com/wwarthen/fd/releases/latest
